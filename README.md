# How to experience it
At first, this mod is **1.21.1-neoforge**<br>
1.Change the type to **Superflat** when creating the world<br>
2.Change the **Preset code format** to **"10*minecraft:stone"**<br>
(If you want, you can change to more layer)<br>

# 怎么体验
首先，这个mod是**1.21.1-neoforge**的<br>
1.创建世界时将世界类型改为**超平坦**<br>
2.将 **预设世界格式码** 改成 **"10*minecraft:stone"**<br>
（如果你想，你可以改变层数）

# What do it do?
In the vanilla version, it was difficult for us to survive in a world with only stones.<br>
However, this mod has changed these.<br>
At the beginning, we will receive a hammer.<br>
Do you want moss? Soak the gravel with a rainwater collector.<br>
Do you want an iron ingot? Use a sieve to remove iron from the gravel!<br>
There are more to come, such as electricity!

Even if you only want ordinary survival, this mod can provide some convenience for it.

# 他干了什么？
在原版，我们很难在一个纯粹的石头世界生存。<br>
但是，这个模组改变了这些。<br>
在开局，我们会得到一把锤子。<br>
想要苔藓？尝试用雨水收集器浸润沙砾。<br>
想要铁锭？用筛子筛沙砾出铁！<br>
后面还有更多，比如电力！

就算你只想要普通的生存，这个模组也能为其提供部分便利。
